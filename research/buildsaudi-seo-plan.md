## Pages to Build

### 1. Company profile pages (automatic SEO magnets)

**URL pattern:** `/companies/[slug]`
**Examples:**
- buildsaudi.co/companies/tamara
- buildsaudi.co/companies/foodics
- buildsaudi.co/companies/tabby
- buildsaudi.co/companies/sary
- buildsaudi.co/companies/cognna

**SEO value:** captures "[company] jobs" and "[company] careers" searches. these are the highest intent keywords because someone is already looking for that specific company.

**meta title pattern:** `[Company] Jobs & Careers — Open Roles at [Company] | BuildSaudi`
**meta description pattern:** `Browse [X] open jobs at [Company]. [One line about the company]. Apply directly on BuildSaudi, Saudi Arabia's curated startup jobs board.`

**target: 50+ company pages at launch, growing to 200+**

### 2. City pages

## Technical SEO Checklist

### must have at launch
- sitemap.xml with all pages (auto-generated by next.js)
- robots.txt allowing all crawlers
- meta title + description on every page (unique, keyword rich)
- open graph tags for social sharing (og:title, og:description, og:image)
- canonical URLs on all pages
- structured data / JSON-LD for job postings (google for jobs integration)
- mobile responsive (google mobile first indexing)
- fast page load (next.js SSG/ISR helps here)
- internal linking between related pages (job → company, company → sector, sector → city)

### google for jobs integration (critical)
google has a special job search feature. if you add JobPosting structured data to each job listing, your jobs can appear directly in google search results. this is massive free traffic.

required schema:
```json
{
  "@context": "https://schema.org/",
  "@type": "JobPosting",
  "title": "Senior Software Engineer",
  "description": "...",
  "datePosted": "2026-03-14",
  "validThrough": "2026-04-14",
  "employmentType": "FULL_TIME",
  "hiringOrganization": {
    "@type": "Organization",
    "name": "Tamara",
    "sameAs": "https://tamara.co"
  },
  "jobLocation": {
    "@type": "Place",
    "address": {
      "@type": "PostalAddress",
      "addressLocality": "Riyadh",
      "addressCountry": "SA"
    }
  }
}
```

### week 1 SEO actions
1. submit sitemap to google search console
2. submit sitemap to bing webmaster tools
3. set up google analytics (or plausible for privacy)
4. create google business profile for buildsaudi
5. add structured data to all job listings

## Arabic SEO (Competitive Advantage)

most english language job boards completely ignore arabic SEO. this is a massive opportunity.

### what to do
- every page should have arabic meta tags (hreflang="ar")
- create arabic versions of the highest value pages:
  - `/ar/jobs/riyadh` — وظائف شركات ناشئة الرياض
  - `/ar/jobs/remote` — وظائف عن بعد شركات ناشئة
  - `/ar/salary/software-engineer` — راتب مطور برمجيات شركات ناشئة
- arabic blog posts targeting وظائف تقنية, شركات ناشئة, رواتب
- this can wait until month 2 but plan the URL structure now
